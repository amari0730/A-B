Calculations

Misclick: Chi-Squared Test

Chi^2: 0.00912

P-Value: 0.923

Degrees of Freedom: 1

For this test I Choose the Chi-Squared test because this is more of a classification task. In other words the user can belong to one of two categories, either correct click or incorrect click. The difference between version A and B is definetly staticically insignificant. The important value is the p value is .923. Thus, I would say that this particular result fails to reject the null hypothesis, and Website A does not have significantly more mislicks than website B.

Time on Page: one-Tail-T-test

T-Score: 3.011

P-Value: 0.998

Degrees of Freedom: 34.26

Avg(B): 10.5

Avg(A): 16.5

Variance(A): 180.5

Variance(B): 144.5

The reason that I chose the One-Tail-T-test for this particular metric is because it is a continuous scenario and it is important to see whether users who used Website A is took longer than Website B, and is dependent on the time interval, which is not categorical. The difference between these two pages is definetly staticically significant. I determined this based on the y value which is approximately .998 which 1 - .998 is less than .05, stating that users who used website A did take signifcantly more time than website B, ultimately rejecting my null hypothesis.
